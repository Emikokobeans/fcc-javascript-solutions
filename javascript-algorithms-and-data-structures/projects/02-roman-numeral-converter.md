## Roman Numeral Converter

Convert the given number into a roman numeral.

All roman numerals answers should be provided in upper-case.

---

### Examples

```javascript
convertToRoman(2);
// should return II
```

```javascript
convertToRoman(83);
// should return LXXXIII
```

```javascript
convertToRoman(798);
// should return DCCXCVIII
```

```javascript
convertToRoman(3999);
// should return MMMCMXCIX
```
